Основы верстки и веб-программирования

Автор курса: Хворостов Владимир Александрович, ассистент кафедры Экономической информатики.


Данный курс предназначен для очного изучения на 1-м курсе магистратуры факультета бизнеса специальности 230700.68 Прикладная информатика в веб-дизайне.

Лабораторная работа №1

В ходе лабораторной работы будет выполнен опредленный список задач.

Задачи:

  • Скачать и установить текстовый редактор Notepad++
  • Создать HTML-документ с произвольным текстом. Документ должен иметь правильную структуру - в нем должны быть правильно использованы теги:
    1. doctype
    2. html
    3. head
    4. title
    5. meta (charset)
    6. body
  • На произвольном тексте по усмотрению студента научиться использовать базовые теги для разметки текста:
    1. p
    2. ul
    3. ol
    4. li
    5. h1-h6
    6. strong
    7. b
    8. em
    9. i
    10. hr
    11. pre
    12. br
  • Научиться вставлять в текст изображения с помощью тега img.
  • Научиться создавать формы и таблицы с помощью тегов form, input, select, textarea, table, tr, td.
  • Создать второй HTML-документ и связать его с первым взаимными гиперссылками (тег a) a.html
  • Провести валидацию своего HTML-документа с помощью http://validator.w3.org/
